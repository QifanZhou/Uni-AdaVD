# Datasets

This folder centralizes the public evaluation prompt files used by the released Uni-AdaVD scripts.

- `i2p_benchmark.csv`: I2P benchmark prompts for implicit unsafe-image evaluation.
- `coco_30k.csv`: Full COCO-30k prompt file used for preservation evaluation.
- `mma_diffusion_nsfw_adv_prompts.csv`: MMA-Diffusion adversarial NSFW prompt set.
- `ring_a_bell/Nudity_eta_3_K_{16,38,77}.csv`: Ring-A-Bell adversarial prompts for nudity-related safety evaluation under three token-length settings.
- `safesora/safesora_*.json`: SafeSora prompt splits used by the ZeroScope public script.

The original local source files are kept outside this folder. These copies are provided to make the public repository layout self-contained.
